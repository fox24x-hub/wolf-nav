# Степной Волк 2026 — Навигатор

Дорожный гид и навигатор для поездки на трейловый забег «Степной Волк 2026».  
Маршрут: Екатеринбург → Оренбург, 15–18 мая 2026.

## Что умеет

- Интерактивная карта маршрута с точками остановок
- GPS-трекинг и отображение скорости в режиме движения
- Карточки-гиды по каждой остановке: что смотреть, сколько времени, фраза для дороги
- Работает офлайн (PWA, Service Worker кеширует тайлы карты и статику)

## Файлы

- `index.htm` — основное приложение (карта + карточки остановок)
- `sw.js` — Service Worker для офлайн-работы

## Технологии

- [Leaflet](https://leafletjs.com/) — карта
- [CartoCDN](https://carto.com/) — тёмные тайлы
- Vanilla JS, без фреймворков
